Export / Import

Every table has field (TableInfo field) containing DateTime stamp when table has been updated.
This field is useful to compare two tables, for example, two tables table1 and table2 are equal if they have
the same name and the same DateTime stamps.

When tables are exported by using old data format or CSV format, then DateTime are not saved.
Therefore, when those tables are imported, then the new DateTime is assigned for every imported table.
Only in case of importing tables from files in Json format the saved there DateTime are restored for
every table.
